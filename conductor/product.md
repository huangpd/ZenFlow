# Product Guide

## Initial Concept
ZenFlow (ZenPath) is a spiritual companion application designed to support users in their journey towards mindfulness and self-discovery. By leveraging Google's Gemini API, it provides personalized guidance and interactive experiences.

## Vision
To create an accessible, AI-powered sanctuary that helps users cultivate peace, clarity, and spiritual growth in their daily lives.

## Target Audience
- Individuals interested in mindfulness, meditation, and spiritual exploration.
- Users seeking a digital tool to track their personal growth and daily practices.
- People looking for AI-assisted emotional or spiritual support.

## Key Features
- **AI Spiritual Companion:** Uses Google Gemini to offer personalized insights, meditation prompts, and conversation.
- **Progress Tracking:** Utilizes local storage to track user milestones and journey continuity.
- **Interactive Experience:** Engaging UI elements (e.g., celebration effects) to encourage user participation and achievement.
- **Privacy-Focused:** Runs locally with local database storage, ensuring user data privacy.

## Platform
- **Web Application:** Built with React, TypeScript, and Vite for a fast, modern web experience.
